# External SQL Injection Resources

Here is a collection of valuable external resources for further learning about SQL injection techniques and defenses:

## Cheatsheets and References

- [Pentestmonkey Cheatsheet](http://pentestmonkey.net/cheat-sheet) - Comprehensive SQL injection cheatsheet
- [MS Access Cheatsheet](http://nibblesec.org/files/MSAccessSQLi/MSAccessSQLi.html) - Specialized for Microsoft Access
- [Reiners Filter Evasion Cheatsheet](https://websec.wordpress.com/2010/12/04/sqli-filter-evasion-cheat-sheet-mysql/) - MySQL filter evasion tactics

## Prevention and Defense

- [OWASP Prevention Cheatsheet](https://www.owasp.org/index.php/SQL_Injection_Prevention_Cheat_Sheet) - Best practices for preventing SQL injection
- [Bobby Tables Prevention Guide](http://bobby-tables.com/) - Practical advice for safe SQL handling
- [HTML5 Security Cheatsheet](http://html5sec.org/) - General web security reference
